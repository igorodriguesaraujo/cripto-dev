# Projeto Cripto.dev

Projeto de estudo desenvolvido com React + consumo de API (API REST) cripto
- https://api.coincap.io/v2/assets/

### Tecnologias

- Typescript
- React
- React Router

Para rodar o projeto use os comandos abaixo

```bash
npm i
```

```bash
npm run dev
```
